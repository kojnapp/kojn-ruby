# Kojn

Kojn's ruby API library.

Feel free to fork, modify & redistribute under the MIT license.

## Installation

Add this line to your application's Gemfile:

    gem 'kojn'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install kojn
